# Generated TATKAL Frontend

This package contains only the frontend.

- The design, styles, labels, and page flow match the approved template.
- Login, register, seat lock, booking, payment order, and history all call your main backend APIs.
- The generated `src/generated-config.js` file already contains the `clientId` and backend URL.

## Run locally

1. Install Node.js 18+.
2. Run `npm start`.
3. Open `http://localhost:4173`.

## Important

- Keep your backend running and accessible from the browser.
- If admin disables or deletes this client ID, backend calls will stop working automatically.
